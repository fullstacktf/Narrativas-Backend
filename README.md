# Endpoints

## /auth/register 

**Method:** POST

**Receives** 

```json
{   
    "username": string,
    "password": string,
    "email": string
}
```

**Returns**

* 201 Created
* 400 Bad Request, { "error": msg }
* 422 Unprocesable Entity, { "error": msg }


## /auth/login 

**Method:** POST

**Receives** 

```json
{   
    "username": string,
    "password": string
}
```

**Returns**

* 200 Ok, { "token": string }
* 400 Bad Request
* 422 Unprocesable Entity
